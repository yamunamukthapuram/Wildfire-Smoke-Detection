## 🌟 About Wildfire Smoke Detection

Wildfire Smoke Detection is an AI-powered web application that automatically detects wildfire smoke from uploaded images using a Convolutional Neural Network (CNN). The platform assists environmental agencies, forest departments, and researchers by providing accurate smoke detection, enabling early intervention and helping minimize environmental damage.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yamunamukthapuram/Wildfire-Smoke-Detection.git
```

### Navigate To Project

```bash
cd Wildfire-Smoke-Detection
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
